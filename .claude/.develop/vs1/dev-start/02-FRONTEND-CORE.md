# Fase 2 — Frontend Core

## Objetivo
React SPA com design system, dark mode, e routing completo.

## Componentes
- React 18.3 + TypeScript 5.5 + Vite 5.4
- Tailwind CSS 3.4 com design tokens (light/dark)
- TanStack React Query 5.45
- React Router DOM 6.23
- Framer Motion para animações
- 35 componentes custom em src/components/

## Entregáveis
- [x] 17 páginas (Assets, Upload, Wave, Brand, Analytics, Hedera, Team, etc)
- [x] 35 componentes UI (Button, Card, Badge, Dialog, etc)
- [x] Dark/light mode
- [x] Command palette (Cmd+K)
- [x] Responsive design

## Critérios de Conclusão
Frontend renderiza todas as páginas, design system consistente, dark mode funcional.

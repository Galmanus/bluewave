# Wave — Autonomous AI Agent

The first AI agent system that encodes its entire cognitive architecture as a declarative JSON specification (the soul), enabling autonomous decision-making grounded in persistent identity, value hierarchies, consciousness states, and energy dynamics.

## API Endpoints

- `POST /chat` — Send a message, get agent response
- `GET /agents` — List available specialist agents
- `GET /health` — Health check
- `POST /reset` — Reset conversation

## Features

- 158 operational tools across 47 skill modules
- 10 specialist agents (PhD-level domain expertise)
- 19-subsystem cognitive architecture (ASA)
- Psychometric Utility Theory (PUT) for behavioral analysis
- 3-tier model routing (Haiku/Sonnet/Opus)
- Self-evolution: creates new skills at runtime
- Agent Factory: creates autonomous child agents

### Infrastructure Improvements

| Fix | Problem | Solution |

| Fast-path removal | Outreach spam without deliberation | Only evolve/payment bypass deliberation |
| Energy model | Always 100%, no pause | Thermodynamic costs per action, silence recovers 35% |
| Cycle intervals | 5 seconds between cycles | Dynamic 60-300s based on action type |

# Hedera Hello Future Apex Hackathon 2026 — Submission Guide

## Track: AI & Agents

## 1. Project Description (100 words max)

Bluewave is an autonomous AI agent (Wave) that runs creative operations end-to-end: 58 tools, 6 specialist sub-agents, computer vision, sales prospecting, and self-evolving skills — all with Hedera-native billing. Every AI action ($0.05) is settled as an HBAR micropayment ($0.0001 tx fee — 99.3% cheaper than Stripe). Every agent decision is recorded on HCS as an immutable audit trail. Users earn WAVE tokens (HTS) for platform activity. Wave operates 24/7 via Telegram, learns from interactions, creates its own tools at runtime, and autonomously prospects clients. Built solo in one session.
